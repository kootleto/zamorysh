**Описание игры**

**_zamorysh_** - это игра про жизнь первокурсника-перфекциониста на ФиКЛе. Игрок занимается разными активностями,
которые
влияют на
усталость, ментальное здоровье, сонливость,
харизму, знания и количество денег.
Можно перемещаться по локациям (дом, парк, кофейни, метро, клуб, университет, улица), где доступны свои активности.
При определённых условиях запускаются автоматические сценарии, которые влияют на игровое состояние.
В игре есть 5 преждевременных концовок и 2 концовки по итогам экзамена.

**Структура репозитория**

- в папке assets лежат изображения, музыка и шрифт
- в папке content лежат активности и сценарии, объединённые по темам, квесты, мини-игры и концовки
- в папке docs лежат гайды по работе с Гитхабом и изображения
- в папке engine лежит движок
- в папке gameplay лежат системы и статы
- в папке interface лежат графический интерфейс (Kivy) и интерфейс командной строки
- в папке tools лежат другие технические вещи
