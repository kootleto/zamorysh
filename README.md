Проект Вероники Вормсбехер, Ани Николаевой, Мити Погорелова, Аксиньи Тамариной и Аси Шипиловой

Это игра про жизнь первокурсника-перфекциониста на ФиКЛе. Игрок занимается разными активностями, которые влияют на
усталость, ментальное здоровье, сонливость,
харизму, знания и количество денег.
