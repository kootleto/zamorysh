Перед запуском игры вы можете настроить параметры запуска в `.env`. Флаги командной строки имеют приоритет над параметрами в `.env`.

## Запуск во встроенном терминале PyCharm

Откройте терминал в PyCharm и выполните команду:

```bash
py run.py
```

*(Windows)*

```bash
python3 run.py
```

*(Linux / macOS)*

Вы можете настраивать параметры запуска не только в `.env`, но и с помощью флагов. Например, укажите после команды запуска (в той же строке через пробел) `--log` или `-l`, чтобы включить логирование, или `--no-log`, чтобы его отключить. Короткие однобуквенные флаги можно комбинировать: `-dl` — включить в игру демо-контент и включить логирование. Если вы не укажете флаги, игра запустится с настройками из `.env`.

> [!tip]
> Полный список флагов можно посмотреть, запустив игру с флагом `--help`.

## Запуск в системном терминале

1. Нажмите правой кнопкой мыши по файлу `run.py`.
2. Выберите `Open In -> Open In Associated Application`.
3. Если в системе настроено открытие файлов `.py` через терминал, игра запустится в системном терминале.
